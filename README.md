# arg_analysis

需要提供两个库：gggenes_mge_preparation.txt   &   gggenes_mge_preparation.txt
其中，两个库都必须含有contig	gene	start	end	strand 五项数据
脚本会自动检索两个库，生成gggenes的准备文件

使用：
# Rscript arg_analysis.R NDM-1
#标黄部分为希望搜索的基因（ARG），arg_analysis.R为脚本的名称
